Useful HTML Tags:

So, far I can't say I have enough experience writing HTML documents to know what the most important tags are. But here is a list of 10 tags I believe are fundamental for most websites:

  1. html - the root element in which all other HTML elements are placed.
  2. head - for adding all important metadata to the HTML document.
  3. body - for adding every HTML element you want to display to the user.
  4. a - for creating hyperlinks to other pages, or to elements within the same page.
  5. link - for linking in external resources like CSS stylesheets to the HTML document.
  6. h1 - for providing a description of the page to search engines.
  7. ol/ul and li - for creating order and order lists of other elements.
  8. img - for displaying images.
  9. vid - for displaying videos.
  10. script - for adding interactivity to the HTML document.
